# Market Pulse: Five-Day Timetable

## Project

Build a small AI finance news app.

The app will take real company news, ask an AI model to analyse it, and present the results in a simple interface. The goal is to learn how a small AI product is put together: data in, model call, structured result, user-facing app.

On Friday, you will demo the project and explain what you built, how it works, and what you would improve next.

## Logistics

- Time: 10am-5:00pm each day
- Location: 16 St John's Ln, London EC1M 4BS

---

# Monday: Understand the Data

Get familiar with the news data and write Python code to load, inspect, and filter it.

You will learn: how Python reads real data, and how to shape that data for the rest of the project.

---

# Tuesday: Make the First AI Call

Send one real headline to an AI model and get a useful response back.

You will learn: how Python talks to an AI model, and how to handle API keys safely.

---

# Wednesday: Make the AI Output Reliable

Ask the model to return a clear, predictable structure instead of loose text.

You will learn: why structured AI output matters, and how to check that the model returned what you asked for.

---

# Thursday: Build the App

Create a simple interface for browsing the analysed news by company.

You will learn: how to turn scripts into an app, and how to present AI results clearly.

---

# Friday: Polish and Demo

Tidy the project, improve the app, and prepare the final demo.

You will learn: how to explain a technical project, and how to judge what worked well or needs improvement.
